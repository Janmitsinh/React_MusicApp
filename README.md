---

## Install dependencies
  Backend

        cd backend
        npm install

  Frontend

        cd ../frontend
        npm install

## Run the project

  Development (concurrently runs frontend + backend)

    npm run dev
  
  Or run separately:
  # In backend/
    npm start

  # In frontend/
    npm start
